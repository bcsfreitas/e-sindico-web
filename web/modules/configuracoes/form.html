Configurações do Sistema de Mensagem

O número do telefone deve conter o código do país, no caso do Brasil esse número é o 55, composto do código de DDD e o número final do celular.

Nome exibido para os usuários que irão receber as mensagens.

Caso escolhido o SMS uma mensagem de texto contendo o código deverá chegar no celular. Caso seja escolhido mensagem de voz uma ligação será efetuada para o celular contendo uma mensagem com o código.


O código de liberação é necessário para o envio e recebimento de mensagens, caso ainda não tenha esse código clique em solicitar código.

Faça um teste enviando uma mensagem para alguém que também utilize o sistema de mensagem .
